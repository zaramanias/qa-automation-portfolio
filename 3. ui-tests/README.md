# ENGLISH
## UI Automation Test Projects

This repository contains several practice projects for **UI test automation** using **Python, Selenium WebDriver, Pytest, and Allure**.

The repository includes automated tests for different web applications:

- **Calculator UI tests** — testing a slow web calculator with delayed calculations  
- **Shop UI tests** — testing the main purchase flow in an online shop  
- **Labirint UI tests** — automating basic user actions on an online bookstore website  

All projects use the **Page Object Model**, Selenium explicit waits, and **Allure reports**.

Each project is located in a separate folder and contains its own detailed `README` with test descriptions and instructions for running the tests.

# РУССКИЙ
## UI авто тесты

Этот репозиторий содержит несколько учебных проектов по **автоматизации UI-тестирования** с использованием **Python, Selenium WebDriver, Pytest и Allure**.

В репозитории представлены тесты для различных веб-приложений:

- **Calculator UI tests** — проверка работы веб-калькулятора с задержкой вычислений  
- **Shop UI tests** — проверка пользовательского сценария покупки в интернет-магазине  
- **Labirint UI tests** — автоматизация базовых пользовательских действий на сайте интернет-магазина  

Во всех проектах используется **Page Object Model**, явные ожидания Selenium и отчёты **Allure**.

Каждый проект находится в отдельной папке и содержит собственный подробный `README` с описанием тестов и инструкцией по запуску.
