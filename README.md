# QA Automation Portfolio

This repository contains examples of software testing including **UI automation testing, API testing, and database validation**.

The projects demonstrate practical QA skills using **Python, Selenium, Pytest, Postman, and PostgreSQL**.

---

## Repository Structure

### UI Tests

Automation tests written with **Python, Selenium WebDriver, and Pytest**.

Projects include:

- Calculator UI tests
- Shop UI tests
- Labirint UI tests

Testing approach:

- Page Object Model (POM)
- Automated regression tests
- Allure reporting

---

### API Tests

API validation using **Postman**.

Includes:

- REST API testing
- Request validation
- Response validation
- JSON structure checks

---

### Database Tests

Database verification using **SQL queries in PostgreSQL**.

Examples include:

- Validating data after API requests
- Checking database consistency
- Verifying data created via UI actions

# Портфолио QA Automation

Этот репозиторий содержит примеры тестирования программного обеспечения, включая **автоматизацию UI-тестирования, API-тестирование и проверку базы данных**.

Проекты демонстрируют практические навыки QA с использованием **Python, Selenium, Pytest, Postman и PostgreSQL**.

---

## Структура репозитория

### UI-тесты

Автоматизированные тесты, написанные с использованием **Python, Selenium WebDriver и Pytest**.

Проекты включают:

- UI-тесты калькулятора
- UI-тесты интернет-магазина
- UI-тесты сайта Лабиринт

Подход к тестированию:

- Page Object Model (POM)
- автоматизированные регрессионные тесты
- отчеты Allure

---

### API-тесты

Проверка API с использованием **Postman**.

Включает:

- тестирование REST API
- проверку запросов
- проверку ответов
- проверку структуры JSON

---

### Тесты базы данных

Проверка базы данных с использованием **SQL-запросов в PostgreSQL**.

Примеры включают:

- проверку данных после API-запросов
- проверку целостности данных
- проверку данных, созданных через UI
